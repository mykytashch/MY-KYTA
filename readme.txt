Создайте виртуальное окружение с помощью следующей команды, 
где venv - это имя виртуальной среды (вы можете выбрать любое имя):

python -m venv venv


Активируйте виртуальное окружение, выполнив команду:

venv\Scripts\activate


Чтобы деактивировать виртуальное окружение, выполните команду:

deactivate

Установите зависимости проекта.

pip install -r requirements.txt


Запустите Flask-приложение.

python run.py
